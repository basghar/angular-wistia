Display

This is the top-level form that will be displayed externally. Subforms can be included.

You can choose to display data from one or more forms each time someone views a record. You can see the primary form below as well as all related forms. You can select which forms you want to display in the portal by turning them on, and which fields to display within each one. You can also drag and drop the related forms to change their order.
